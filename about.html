Hakkımızda

NASA Uzay Portalı, uzay ve astronomi alanındaki güncel verileri kullanıcılarla buluşturmak amacıyla geliştirilmiş, mobil uyumlu ve modern bir web uygulamasıdır.

Uygulama; NASA tarafından sunulan açık veri servisleri aracılığıyla günlük astronomi görsellerini, asteroit takip verilerini, uzay hava olaylarını ve NASA Görsel Kütüphanesi içeriklerini tek bir platformda sunar.

Bu proje, gerçek bir kurumsal web sitesi senaryosu temel alınarak tasarlanmış olup; kullanıcı deneyimi, erişilebilirlik ve performans öncelikleri gözetilmiştir.

Teknik Altyapı

  • Kullanılan API’ler: NASA Open APIs
    APOD, NeoWs (Asteroid Radar), DONKI (Space Weather), NASA Image & Video Library
  • CSS Çatısı: Bootstrap 5.3
    Responsive grid sistemi ve hazır UI bileşenleri kullanılmıştır.
  • Geliştirme Yaklaşımı:
    Vanilla JavaScript, modüler dosya yapısı ve API hata yönetimi prensipleri uygulanmıştır.
Offline (Çevrimdışı) Çalışma Mantığı

Bu uygulama Progressive Web App (PWA) standartlarına uygun olarak geliştirilmiştir. Service Worker sayesinde HTML, CSS ve JavaScript dosyaları cache’lenir; internet bağlantısı olmadığında bile daha önce ziyaret edilen sayfalar görüntülenebilir. API erişilemediği durumlarda örnek (sample) veriler otomatik olarak devreye girer.
